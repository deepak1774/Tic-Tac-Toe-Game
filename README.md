# Tic-Tac-Toe Game

A classic Tic-Tac-Toe game built with HTML, CSS, and Vanilla JavaScript.

## Features
- **Responsive Design**: Works on mobile and desktop.
- **Modern UI**: Clean design with smooth interactions.
- **Interactive Gameplay**: Two-player local mode.
- **Status Indicators**: Real-time turn tracking and game outcome announcement.

## Preview
Open `index.html` in any modern web browser to start playing.

## How to Play
1. The game is played on a 3x3 grid.
2. Two players take turns placing their marks ('X' or 'O').
3. The first player to get 3 of their marks in a row (horizontally, vertically, or diagonally) is the winner.
4. If all 9 cells are filled and no player has 3 marks in a row, the game ends in a draw.

## Installation
No installation required! Just clone the repository and open the HTML file.

```bash
git clone https://github.com/deepak1774/Tic-Tac-Toe-Game.git
cd Tic-Tac-Toe-Game
# Open index.html in your browser
```
